What is CSS?

  1. CSS stand for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.

  3. Css is not neccesary CSS saves a lot of work.It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files,

CSS can be added to HTML Document in 3 wayes:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using the style elements in head section
  • External - by using a link elements to link to an External
  • The most common way to add CSS. is to keep the styles in External css files.Howere , in this tutorial we will use inline and External styles, beacuse this is easier to demonstrate , and easier for you to try it yourself

When comparing CSS class vs ID,the difference is that CSS class applies a style to multiple elements. ID , on the other hand ,applies a style to one unique elements.ID is also special in that you can use a special URL,to link directly to an elements and if's used by JavaScript.

In CSS , selectors are used to target a specific element or range of elements on a web page.Onece an element has been targeted, a style or set of style can be applied to the element.

There is a wide range of selectors availble. Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied
